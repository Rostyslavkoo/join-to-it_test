# Calendar App

Calendar application with event management built with Vue 3 + TypeScript + Vuetify.

## Features

- ✅ Add, edit, and delete events (max 30 characters)
- ✅ Color selection for events
- ✅ Drag & drop events
- ✅ Four views: Month, Week, Day, Agenda
- ✅ Proper display of overlapping events
- ✅ LocalStorage persistence

## Tech Stack

- Vue 3
- TypeScript
- Vuetify 3
- Vite
- date-fns

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

Automatically deploys to GitHub Pages on push to `main` branch.
